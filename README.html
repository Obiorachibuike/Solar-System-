# Solar System - Chibuike Obiora ## Features - Real-time 3D simulation of the solar system using Three.js - Planets orbit the sun based on real-like speed and distance - Control panel with sliders for adjusting orbital speed of each planet - Pause/Resume animation - Starry background - Labels/tooltips on hover - Camera zoom on planet click - Dark/Light theme toggle ## How to Run 1. Download the folder. 2. Open `index.html` in your browser. ## Tech Used - Three.js - JavaScript (ES6) - HTML/CSS ## Author Chibuike Obiora
